Help you quick create many topbars.

== Description ==

W3Topbar allows you create and manage one or many topbars which are displayed on specify in your website.
The place may page, post, custom post type, archive, taxonomy or any where you want. Of cource, you can disable it if you want.

A page can have many topbars. But a topbar only is displayed in individual page.

Every topbar has many options allows you customize it. If you wanna reset all your changes, so you may easier do this by click "Reset all button" in settings page.

Once you delete the plugin, the deleting process will remove entire tracks in your database.

== Installation ==

You can install the plugin by two ways:

1. Through Wordpress admin panel:
	Goto "Plugins -> Add new". Type "w3topbar" into search box.
	Then click "Install Now" button and active it.
	* if you have one, just click "Upload plugin" on the top and choose the plugin zip file.
2. Upload through FTP
	Upload plugin folder into plugin directory.
	Active it in admin.
